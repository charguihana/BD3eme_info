Cours de Manipulation SQL

Notion 1 : Consultation Afficher toutes les données
SELECT * FROM eleve;
Leçon :

Afficher le contenu actuel de votre table eleve.

L'étoile * signifie "tout". On demande à MySQL d'afficher toutes les colonnes présentes dans la table.
Notion 2 : Tri Trier les résultats

Organisons maintenant les élèves par ordre alphabétique sur leurs noms.

SELECT * FROM eleve ORDER BY nom ASC;

Organisons les élèves par ordre alphabétique sur leurs noms.

Leçon : ORDER BY permet de trier. ASC est pour l'ordre croissant (A-Z) et DESC pour l'ordre décroissant (Z-A).
Notion 3 : Recherche Recherche par motif (Le LIKE)
SELECT * FROM eleve WHERE nom LIKE 'A%';

Cherchons tous les élèves dont le nom commence par la lettre 'A'.

Leçon : LIKE est utilisé pour chercher des modèles de texte. Le % remplace n'importe quelle suite de caractères.
Notion 4 : Dates Filtrer par mois
SELECT Nom, Prenom FROM eleve WHERE MONTH(date_naissance) = 3;

Extraire les nom et les Prenom des élèves nés durant un mois précis (ici Mars).

Leçon : La fonction MONTH() permet d'extraire uniquement le numéro du mois d'une date (1 pour janvier, 3 pour mars, etc.).
Notion 5 : Modification Mise à jour des données
UPDATE eleve SET ville = 'Tunis' WHERE id_eleve = E002;

Modifions la ville de l'élève qui possède l'ID numéro E002.

Leçon : UPDATE modifie des données existantes. Le WHERE est indispensable, sinon tous les élèves habiteraient à Tunis !
Notion 6 : Suppression Supprimer un enregistrement
DELETE FROM eleve WHERE id_eleve = E005;

Supprimons définitivement l'élève numéro 5 de la base.

Leçon : DELETE supprime une ligne. C'est une action irréversible. On utilise l'ID pour être sûr de supprimer le bon élève.
